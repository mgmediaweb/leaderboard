# Leaderboard

> this application lists the scores of players of a certain game (determined by the user), in this app the Leaderboard REST API is used, where the participants and their scores are registered.


## Built With

- HTML5
- CSS3
- JAVASCRIPT ES6
- API REST
- WEBPACK
